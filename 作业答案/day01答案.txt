
0.      什么是iOS，什么事mac os，什么是os x？
		ios:搭载在苹果的硬件上，由苹果公司开发的操作系统。
		mac os:苹果公司开发的pc（个人电脑)操作系统。
		os X:是指mac os 的第10个版本，X饰罗马数字的10，读作ten。

1.      什么IDE？
	集成开发环境。

2.	C程序的源文件拓展名是什么？
	.c

3.	C程序的源文件能直接运行么？为什么？
	不能，需要进行编译、链接两个阶段之后才能运行。

4.	C程序从开发到运行需要哪些步骤？
	需要经过：编写->编译->链接->运行。

5.	拓展名为.c、.o、.out分别是什么文件？在C程序开发的哪个阶段产生的？
	“.c”是C语言的源文件，在编写程序阶段产生。
	".o"是目标文件，在编译后产生.
	".out"是生成的可执行文件，在链接后产生。

6.	下面这段程序是否能独立运行成功？

#include <stdio.h>
int test()
{
	printf("哈哈哈哈\n");
	return 0;
}
答：不能，因为缺少主函数，主函数是程序的入口，因此缺少执行入口，不能独立运行。

7.	下面这段程序能否运行成功？
int main()
{
	printf("哈哈哈哈\n");
	return 0;
}
答：可以。
